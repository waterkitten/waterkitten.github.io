补充：token 可以译做通证，也可以译做代币

NFT 是 非同质化通证（non-fungible token）, 好似说了又好像没说。本文将抽象地给出 NFT 的含义，同时也给些了一些 NFT 应用程序和用例的具体例子。

首先，让我们来分析一下这个名字。在我们讨论 “非同质化” 之前，我们所说的 “通证”（token）是什么意思？对于我们来说，一个通证（token）是信息系统中称为区块链的一种记录。我们这里不会详细讨论区块链是如何工作的。要理解 NFT 是什么，我们只需要先来补充一些背景知识。

# 一点历史补充

最着名的区块链创造了该该术语。比特币的白皮书 引入了一个 "区块链" 的概念，在比特币形成过程中，区块是一个一个的存储单元，记录了一定时间内各个区块节点全部的交流信息。各个区块之间通过随机散列 (也称哈希算法) 实现链接，后一个区块包含前一个区块的哈希值，随着信息交流的扩大，一个区块与一个区块相继接续，形成的结果就叫区块链。比特币网络有一个名为 "比特币" 的代币（Token）。

像传统现金一样，比特币是可替代的（fungible），这意味着你可以用另一个比特币替换一个比特币，而不会改变任何一个比特币的价值。事实上，在技术层面上，并没有像 "单个比特币" 这样的东西有自己的身份标识。（这里可能有些拗口，因为定义上 NFT 就是 non-fungible），虽然现金至少可能有一个序列号来区分一张钞票和另一张钞票，但比特币网络不会单独跟踪每个比特币并为它们分配一个标识符。相反，系统只是跟踪每个帐户中的数量，每笔交易都会贷记一个帐户并借记另一个帐户。

比特币启发了许多其他区块链，这些区块链迭代了相同的基本思想并引入了新功能。而这些迭代中最有标志性意义的就是以太坊对于 NFT 的概念的开发。以太坊在比特币开创的区块链共识模型中增加了通用计算（general-purpose computation），将自己定位为 "世界计算机"（world computer），实现 "可编程货币"（"programmable money”）

以太坊有一个名为 Ether 的原生代币，它既被用作价值储存，又以称为 gas 的费用形式支付计算费用。像比特币一样，以太币是可替代的（fungible）。但是，智能合约计算模型允许开发人员创建自己的通证（token），这些通证（token）可以根据合约中的逻辑具有特殊属性。

大多数早期的智能合约定义了类似于以太币的同质化代币，但人们很快开始尝试使用通证来包含数据（哈希），使每个代币都独一无二，并且可以与其他代币区分开来。结果是无法与另一个相同类型的任意代币或非同质化代币去做简单交换.

# 是什么使通证不可替代（非同质化）？

早些时候，我们提到序列号是区分一美元钞票与另一张钞票的一种方式。即使你可以区分它们，两美元钞票仍然是可相互替代的，因为它们每个都与货币具有相同的价值。仅凭唯一标识符不足以使某些内容不可替代。

然而，毕加索绘制的美元钞票是独一无二的，与序列号无关。通过将其用作艺术品的媒介，我们想象中的毕加索使账单的可替代性降低。虽然从技术上讲，你可以用它换取价值一美元的零食，但现在钞票的交换价值远远大于一美元。

同样，通过允许每个令牌包含少量任意数据，NFT 成为创造性表达的媒介，以及交换和记账单位。因此，NFT 的值高度依赖于它所包含和表示的意义。同一个 NFT 可能被不同的人完全不同地对待，这取决于审美品味或创作者的身份等因素。

当然，我们不通过交换无价的毕加索素描来开展业务是有充分理由的！如果没有同质性，NFT 就不如货币（不能像货币那样可以随时在市场流通）。不过，NFT 之间通过存在于支持数字货币的相同网络上，NFT 可以利用支付和账户基础设施来完成交易，并从区块链的安全保证中受益。

# NFT 有何特别之处？

是什么让 NFT 与某人网站上的动画 GIF 文件或其他类型的数字记录（如电子表格）不同？

这些差异取决于区块链设计的一些核心属性。区块链网络的主要功能是让所有参与者就一个共享的 "世界状态" 达成一致。对于比特币，共享状态是每个账户的余额，而对于以太坊，共享状态是智能合约交互的输入和输出。由于网络成员分布在世界各地，因此每个人都需要时间才能汇聚在同一状态上，并且有特殊的规则来防止作弊或恶意行为。一旦每个人都同意了状态，它就成为区块链规范历史的一部分。

随着新区块被添加到链中，早期的区块变得更难修改，修改成本也更高。很快 —— 在几个区块之间的 "change history（修改历史）" 将会大到无法更改，因此区块链中记录的信息可以被认为是永久性的。

相比之下，传统的网络是众所周知的无常和动态的。例如，Web 服务器可能会根据一天中的时间或访问者 IP 地址的地理位置提供不同的内容。正如我们日常面对的失效链接（404）一样，内容从网络上消失和生产的频率一样频繁。

这种持久性和稳定性的特性是 NFT 价值主张的核心。通过使用区块链作为持久的共享数据存储介质，只要区块链本身保持运行，NFT 就可以被信任地持续下去 - 这带来了区块链的另一个有趣的特性。通过用加密货币奖励节点运营商，以换取保持网络的活力，区块链激励了自己的生存。只要有人被经济回报所吸引，就会有人有动力保持网络在线。这确保了所有历史数据的存续，包括 NFT。

# NFT 的用途是什么？

最早的 NFT 实验是 CryptoPunks（加密朋克），一组 10，000 像素的艺术角色，可以在以太坊上收集和交易。虽然任何人都可以查看任何朋克的艺术作品，但每个角色在给定时间在以太坊区块链上只能有一个官方 "所有者"。CryptoPunks 被证明是一个巨大的成功，罕见的朋克交易手，价值数百万美元的巨额以太币。

像 CryptoPunks 这样的数字收藏品仍然是 NFT 最受欢迎和最引人注目的用例之一，CryptoPunks 的继任者帮助定义了 NFT 互操作性和元数据的最早标准。

自 CryptoPunks 首次亮相以来的几年里，NFT 已成为各种创意项目的工具。大部分兴奋来自艺术家，他们希望直接与粉丝互动，并找到以他们的作品谋生的新方式。这得到了不断增长的 NFT 市场生态系统的支持，这些生态系统弥合了熟悉的网络和神秘的加密货币世界之间的差距，并且还导致价值数百万美元的直接支付给艺术家的数字创作。

在数字艺术和收藏品之外，NFT 已经在游戏中找到了用例，它们可以代表虚拟土地的情节，游戏角色的化身和皮肤，游戏内物品等。通过将玩家将这些物品的 "库存" 放在共享的区块链上，NFT 驱动的游戏可以启用新的机制，并允许玩家在多个潜在的游戏和体验中使用他们的自定义物品。它们还为玩家提供了一种在他们之间购买，出售和交易物品的方式，而无需将购买的物品锁定在单个公司的店面或市场中。

# 关于 Cryptopunks 的补充介绍

10,000 个独特的收藏字符，并带有存储在以太坊区块链上的所有权证明。激发现代 CryptoArt 运动的项目。

Cryptopunks 是以太坊上最早的 “NTF 不可替代代币” 的例子之一，也是支持大多数数字艺术和收藏品的 ERC-721 标准的灵感来源。

CryptoPunks 是 10,000 个唯一生成的字符。没有两个是完全相同的，并且每个都可以由以太坊区块链上的一个人正式拥有。最初，任何拥有以太坊钱包的人都可以免费领取它们，但很快就全部 10,000 个领取了。现在，它们必须通过嵌入在区块链中的市场从某人那里购买。通过这个市场，您可以购买、竞标和出售朋克。蓝色背景的朋克不出售，目前没有出价。朋克与红色背景可供其所有者出售。最后，紫色背景的朋克对他们有积极的出价。

什么是 CryptoPunks（加密朋克）
CryptoPunks 是 24x24 像素的艺术图像，通过算法生成。大多数都是看起来很笨拙的男孩和女孩，但也有一些比较罕见的类型：猿、僵尸，甚至是奇怪的外星人。每个朋克都有自己的个人资料页面，显示他们的属性以及他们的所有权 / 出售状态。

# 展望未来

我们仍然处于 NFT 的早期阶段，我们很可能会看到一系列有趣的 NFT 用例和体验，这些用例和体验超出了当前我们所做事情的认知。很难预测未来会带来什么，但如果你坚持学习，你比大多数人都有更好的机会。正如他们所说，预测未来的最好方法是自己去创造它！
